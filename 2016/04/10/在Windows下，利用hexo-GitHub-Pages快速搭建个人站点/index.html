安装NodeJs

去nodejs.org下载以.msi为扩展名的安装包，如果不需要进行NodeJs相关开发的话选择LTS版本就够了，双击安装

安装GitHub Desktop

直接下载GitHub Desketop，安装并登录即可

安装hexo

至此，即可使用hexo搭建站点了
